# Service Domains\n\nDomain service directories for the staged major upgrade.\n\n- `auth/`: login, token, WeChat binding, account challenge flow\n- `users/`: profiles, notifications, user requests\n- `devices/`: device listing, detail, admin device management\n- `reservations/`: reservation, precheck, approval, borrowing and return\n- `admin/`: dashboard, permissions, operation logs, security config\n- `chat/`: conversations, messages, SSE events\n- `reports/`: statistics, exports, daily reports\n- `wechat/`: callback handling and outbound WeChat messages\n\nDo not move large business blocks here without a passing `npm run check` checkpoint.\n
